STM32+ESP8266 无线温湿度监测系统
基于STM32F103C8T6 + ESP8266-01S的环境温湿度无线监测系统，支持本地OLED显示、触摸按键切换、蜂鸣器超标报警、TCP透传远程上传。

 核心功能
- DHT11温湿度实时采集
- ESP8266 TCP透传，数据定时上传至PC
- 0.96寸I2C OLED多界面显示
- TTP224 4路触摸按键交互
- 温湿度超标蜂鸣器报警

 硬件平台
STM32F103C8T6、ESP8266-01S、DHT11、0.96寸I2C OLED、TTP224触摸按键、有源蜂鸣器
开发环境
Keil MDK5、STM32CubeMX
